# Trybe
​
Este positório contém todas as atividades de apndizagem desenvolvidas por _[Seu nome](https://www.linkedin.com/in/jean-fonseca-eng/)_ enquanto estudava na [Trybe](https://www.betrybe.com/) :rocket:
​
_"A Trybe é uma escola do futuro para qualquer pessoa que queira melhorar de vida e construir uma carira de sucesso em tecnologia, onde a pessoa só paga quando conseguir um bom trabalho."_
​
O programa conta com mais de 1.500 horas de aulas online ao vivo, aborda introdução ao desenvolvimento de softwa, front-end, back-end, ciência da computação, engenharia de softwa, metodologias ágeis e habilidades comportamentais.
